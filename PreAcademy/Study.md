# MVP (Minimum Viable Product)

: 프로그램이 제공하고자 하는 서비스에서 최소한의, 필수적인 기능만을 구현한 제품.
(다른 방법론들도 많다. ex.PoC)

아래 일기장 리스트에서 최소한의 기능만 선택하면 다음과 같다.

- [x] 일기 쓰기
- 일기 수정하기
- 일기 삭제하기
- [x] 일기 보기
- [x] 달력 UI 제공
- 오늘 보기
- 월 변경
- 년 변경
- 날짜 변경
- 오늘로 이동
